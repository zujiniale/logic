## 🖥️ Introdução à Computação

### Hardware vs Software
| Conceito | Definição |
|----------|-----------|
| **Computador** | Coleção de componentes interligados para processar operações aritméticas e lógicas |
| **Hardware** | Parte **física** do computador (componentes estruturais) |
| **Software** | Parte **lógica** — programas que permitem realizar atividades específicas |

---

## 🧠 O que é Lógica?

### Definições
- **Aristóteles (Lógica):** Estudo das leis do raciocínio para demonstrar a verdade
- **Forbellone (1999):** "Sequência de passos que visa atingir um objetivo bem definido"
- **Ascencio (1999):** "Descrição de uma sequência de passos que deve ser seguida para realização de uma tarefa"

### Tipos de Lógica
| Tipo | Descrição |
|------|-----------|
| **Lógica Aristotélica** | Estuda o pensamento, conceito, juízo e raciocínio |
| **Lógica Matemática** | Sistema dedutivo de enunciados com leis e regras |
| **Lógica de Programação** | Linguagem usada para criar programas de computador |
| **Lógica Proposicional** | Examina relações entre proposições (verdadeiras ou falsas) |

### Por que é importante?
- Requisito fundamental para programar
- Grande diferencial entre programadores
- Permite reconhecer, interpretar e estruturar problemas para solucioná-los

## 🔢 O que é um Algoritmo?

> **Algoritmo** é uma sequência lógica de passos finitos e encadeados que descrevem como um determinado problema deve ser resolvido.

### Exemplo do cotidiano — "Comer uma Bala":
1. Pegar a bala
2. Retirar o papel
3. Colocar a bala na boca
4. Jogar o papel no lixo

### Fases do Algoritmo
```
[ENTRADA] → [PROCESSAMENTO] → [SAÍDA]
```
| Fase | Descrição |
|------|-----------|
| **Entrada** | Dados que serão processados |
| **Processamento** | Manipulação das informações para produzir o resultado |
| **Saída** | Resultado esperado após o processamento |

### Exemplo — Média de Alunos:
- **Entrada:** P1, P2, P3, P4
- **Processamento:** (P1 + P2 + P3 + P4) / 4
- **Saída:** Média Final

---

## 📝 Formas de Representar Algoritmos

### 1. Descrição Narrativa
Passos escritos em linguagem natural.

**Exemplo — Lavar o cabelo:**
1. Molhar o cabelo
2. Aplicar o xampu
3. Massagear
4. Enxaguar

### 2. Fluxograma
Representação **gráfica** onde formas geométricas diferentes representam ações distintas.
- Usado para visualizar o fluxo do algoritmo de forma intuitiva

### 3. Pseudocódigo
Representação semiestruturada próxima à linguagem de programação.

**Exemplo — Algoritmo de Soma:**
```
Algoritmo Soma
Var
  n1, n2, S : Inteiro
Início
  Escreva("Entre com o primeiro valor: ")
  Leia(N1)
  Escreva("Entre com o segundo valor: ")
  Leia(N2)
  S <- N1 + N2
  Escreva("Soma =", S)
Fim.
```

## 🤖 IA vs Programadores

A IA está **redefinindo o papel** dos desenvolvedores:
- Automatiza tarefas rotineiras → aumenta produtividade
- Libera engenheiros para focar em **problemas complexos**: arquitetura, integração, estratégia, criatividade
- O programador moderno se torna um **orquestrador de tecnologia**, não apenas um implementador de código
